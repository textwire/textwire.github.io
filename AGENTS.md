## Information

- This project uses Vitepress with Vue.
- Theme files are located in `docs/.vitepress/theme` directory.
- Syntax highlighting rules are in `docs/.vitepress/textwire.tmLanguage.json` file.
- Vitepress configuration file in `docs/.vitepress/config.mts`.
- All the docs are seperated into versions, they are located in `docs/versions` directory.
- Articles are located in docs/.vitepress/blog directory. The index.md file there is a page to display all articles that I have in `docs/.vitepress/theme/modules/blogPosts.ts file`.
- There are no tests and I don't need them.
- To check any errors you can build a project with `npm run build`.
