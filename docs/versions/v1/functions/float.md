# Float functions

### `int()`

#### Description
Function `int` is used to convert a float to an integer by removing the decimal part of the number. It doesn't round the number, it just removes the decimal part.

##### Arguments
None

#### Input Example
```textwire
<b>{{ 5.5.int() }}<b>
```

#### Output
```html
<b>5<b>
```
