# Get Started

This guide will help you get Textwire up and running in your Go project. You'll learn about the requirements, installation process, and how to create your first template.

## Requirements

Go `1.25` or higher is required to use Textwire. If you don't have Go installed on your machine, you can download it from the [official Go website](https://golang.org/dl/).

## Installation

Install the Textwire package in your Go environment:

```bash
go get -u github.com/textwire/textwire/v3
```
